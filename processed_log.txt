## 1. 날짜별 오류 횟수 (SQL)
질문:
1월 29일 오류 몇 번이야?

분류: sql
예상 SQL:
SELECT COUNT(*) FROM logs 
WHERE Year = 2021 AND Month = 1 AND Day = 29 AND operation = 'error';

예상 답변:
✅ 2021년 1월 29일 발생한 오류 횟수는 총 7회입니다.

## 2. 고장 기계 점검 (manual + SQL)
질문:
오늘 어떤 기계를 점검해야 하고 어떻게 점검해야 해?

분류: manual
흐름:
- 최근 하루 기준 오류+정지 기계 조회
- 매뉴얼 검색
- GPT 요약 및 안내

예상 답변:
🛠️ CNC2
최근 정지 및 오류가 반복적으로 발생했습니다.
- 원인: 냉각 센서 오작동 가능
- 영향: 공정 중 정지 발생
- 조치: 센서 선 확인 → 재결선 → 10분간 공회전 테스트

🛠️ MCT1
이틀 연속으로 오류 발생.
- 원인: 주축 과열
- 조치: 윤활 상태 확인 → 냉각팬 점검

## 3. 시간대별 고장 집중 구간 (SQL)
질문:
하루 중 고장이 많이 나는 시간대는 언제야?

분류: sql
예상 SQL:
SELECT Hour, COUNT(*) as error_count 
FROM logs 
WHERE operation = 'error' 
GROUP BY Hour 
ORDER BY error_count DESC;

예상 답변:
📈 고장 발생이 가장 많았던 시간대는 오전 9시입니다.  
다음으로 10시, 13시 순으로 오류가 집중되어 있습니다.

## 4. 특정 기계 점검 요청 (manual only)
질문:
CNC2 고장났는데 어떻게 정비해?

분류: manual

예상 답변:
🛠️ CNC2
- 원인: 주축 회전 센서 불량 가능
- 영향: 반복적인 정지, 스핀들 과속
- 조치:
  1. 전원 OFF 후 센서 단자 확인
  2. 케이블 연결 상태 점검
  3. 이상 없을 경우 센서 교체 필요
