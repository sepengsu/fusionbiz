# 📦 기계 A - 점검 매뉴얼

## 🔧 [고장 유형: 반복적인 정지 발생]

- 설명: 짧은 시간 간격으로 작동 → 정지 패턴이 반복됨
- 원인 가능성:
  - 센서 오류로 잘못된 동작 감지
  - 부품 마모로 인한 안정적 동작 실패
- 조치:
  1. 센서 점검: 오작동 신호가 지속되는지 확인
  2. 기계 내부 청소 및 마모 부품 교체
  3. 전원 차단 후 재부팅 (5분 이상 대기)
- 🔎 대응 키워드: `반복 정지`, `계속 멈춤`, `센서`, `재부팅`

---

## 🔧 [고장 유형: 오전 시간대 오류 집중 발생]

- 설명: 오전 9~11시 사이에 오류 및 정지 빈번
- 원인 가능성:
  - 초기 구동 시 과전류 또는 과열
  - 실내 온도 변화로 인한 오작동
- 조치:
  1. 전원 켜기 전 10분 예열 권장
  2. 초기 시점에 수동 테스트 모드로 작동 시작
  3. 에러 발생 시 로그 저장 후 관리자 호출
- 🔎 대응 키워드: `아침`, `9시`, `10시`, `초기 구동 오류`

# 📦 기계 B - 점검 매뉴얼

## 🔧 [고장 유형: 에러 → 정지 후 복구 안 됨]

- 설명: 에러 발생 후 정지되고 다시 작동하지 않음
- 원인 가능성:
  - 메모리 누수 혹은 내부 과부하
  - 긴급정지 시스템 작동 중
- 조치:
  1. 메모리 상태 확인 (에러 코드 X21 확인)
  2. 긴급정지 해제 후 관리자 권한으로 재가동
  3. 문제 지속 시 기계 상태 리포트 출력하여 유지보수 의뢰
- 🔎 대응 키워드: `에러 후 멈춤`, `재가동 불가`, `정지 이후 동작 안됨`

# 📦 기계 C - 점검 매뉴얼

## 🔧 [고장 유형: 작동 중 오류 간헐 발생]

- 설명: 작동 중 정상 작동하다 특정 시점에 에러 후 자동 회복
- 원인 가능성:
  - 불규칙한 전류 흐름
  - 간헐적 부품 접촉 불량
- 조치:
  1. 전원선 및 접촉부 이상 여부 점검
  2. 최근 1시간 로그 확인
  3. 동일 패턴 반복 시 부품 교체 고려
- 🔎 대응 키워드: `작동 중 에러`, `중간에 멈춤`, `간헐 정지`
